#### 1. Disk Space Usage

**`df -h`**

- df : report file system disk space usage
- -h : human readable

#### 2. File Space Usage

**`du -sh /home/john`**

**`du -sh Images/*`**

- du : estimate file space usage
- -s : summarize (only display the total space usage)
- -h : human readable
